## [2.1.3] - 2025-01-20

### 🏠 **대시보드 네비게이션 개선**

#### ✨ **OpenManager 버튼 수정**
- **세션 정리 로직 추가**: 대시보드에서 랜딩페이지 이동 시 인증 정보 완전 정리
- **handleGoToLanding() 함수**: localStorage, sessionStorage 모든 인증 정보 제거
- **순환 이동 지원**: 랜딩페이지 ↔ 대시보드 자유로운 이동 가능
- **세션 충돌 방지**: 이전 세션 정보로 인한 인증 문제 해결

#### 🤖 **AI 에이전트 자동 활성화 시스템**
- **대시보드 접근 시 자동 활성화**: 인증 성공 후 시스템 자동 활성화
- **AI 질문 시 자동 활성화**: 메시지 전송 시 시스템 상태 확인 후 자동 활성화
- **데스크탑/모바일 모두 적용**: AgentPanel, AgentPanelMobile 동일 로직 적용
- **절전모드 메시지 제거**: "시스템이 절전 모드입니다" 메시지 완전 제거

#### 🔧 **사용자 경험 개선**
- **즉시 응답**: AI 에이전트 클릭 시 바로 질문 가능
- **투명한 활성화**: 콘솔 로그로 시스템 활성화 과정 확인 가능
- **안정적인 세션 관리**: 페이지 이동 시 세션 상태 정리
- **일관된 동작**: 모든 AI 인터페이스에서 동일한 자동 활성화

#### 🚀 **기술적 개선**
- **PowerStore 활용**: `usePowerStore.getState().activateSystem()` 직접 호출
- **조건부 활성화**: `!isSystemActive` 상태에서만 활성화 실행
- **에러 방지**: 이미 활성화된 상태에서는 중복 실행 방지
- **로깅 강화**: 활성화 과정의 각 단계별 상세 로그

## [2.1.1] - 2025-01-20

### 🔧 **API 완성도 개선 및 404 에러 해결**

#### ✨ **개별 서버 API 라우트 완전 구현**
- **새로운 엔드포인트**: `/api/servers/[id]` 추가
  - `GET`: 개별 서버 상세 정보 조회
  - `PUT`: 서버 정보 업데이트
  - `DELETE`: 서버 제거 (데모 모드에서는 제한)

#### 🔧 **기술적 개선**
- **Next.js 15 타입 시스템 호환**: Promise 기반 params 처리
- **ID 기반 일관된 데이터 생성**: 동일 서버 ID에 대한 일관된 메트릭
- **쿼리 파라미터 지원**: `history`, `hours` 등 옵션 제공
- **에러 처리 강화**: 상세한 에러 메시지 및 로깅

#### 🎯 **404 에러 완전 해결**
- **브라우저 콘솔 에러 제거**: `api/servers/server-001` 요청 정상 처리
- **API 응답 구조 통일**: 일관된 데이터 형식 제공
- **서버 히스토리 데이터**: 24시간 CPU/메모리 히스토리 지원

#### 📊 **데이터 구조 개선**
- **실제적인 서버 메트릭**: CPU, 메모리, 디스크, 네트워크 데이터
- **서비스 상태 정보**: nginx, 애플리케이션 서비스 모니터링
- **알림 시스템**: 임계값 기반 자동 알림 생성
- **태그 시스템**: 팀, 프로젝트, 비용 센터별 분류

#### 🚀 **성능 최적화**
- **빌드 성공**: 36개 라우트 모두 정상 빌드
- **TypeScript 경고 해결**: 사용하지 않는 import 제거
- **코드 품질 개선**: 일관된 에러 처리 및 로깅

## [2.0.0] - 2025-01-20

### 🧠 **AI 에이전트 시스템 완전 재구현**
- **스마트 모드 감지 시스템**
  - `SmartModeDetector.ts`: 질문 유형 자동 분석 (5가지 카테고리 점수 시스템)

- **실시간 사고 과정 시각화**
  - `ThinkingProcessor.ts`: 7단계 사고 과정 실시간 스트리밍
  - `ThinkingDisplay.tsx`: 단계별 진행률 바, 상태 아이콘, 소요시간 표시
  - Server-Sent Events 기반 실시간 업데이트
  - 분석→분류→처리→응답생성→검증 흐름 명시적 표시

- **관리자 대시보드 시스템**
  - `EnhancedAdminDashboard.tsx`: 6개 탭 관리자 대시보드
  - `AdminLogger.ts`: 모든 상호작용 로깅 및 성능 메트릭 수집
  - 다단계 보안 인증 (기본 자격증명 + 2FA TOTP)
  - IP 기반 차단 시스템 (5회 실패시 15분 차단)
  - 복사 방지 기능 (Ctrl+C, 우클릭, F12, 선택 차단)

### 🔐 **보안 및 인증 시스템**

  - 세분화된 권한 관리 (`system:admin`, `ai_agent:read/write`, `logs:export`)

  - 관리자 인증 모달 통합
  - 인증 성공시 `/admin/ai-agent`로 자동 이동

### 📊 **학습 데이터 관리 시스템**
