# History (프로젝트 히스토리)

이 폴더는 프로젝트의 중요한 역사적 기록을 보관합니다.
**삭제 대상이 아닌 영구 보관 문서입니다.**

## 폴더 구조

```
history/
├── completed/     # 완료된 주요 작업 기록
├── planning/      # 구현된 기능의 설계 문서
├── reports/       # 완료된 프로젝트 보고서
└── *.md           # 주요 기술적 분석/결정 기록
```

## 보관 기준

- 프로젝트 의사결정 히스토리
- 완료된 주요 기능/리팩토링 기록
- 트러블슈팅 분석 및 해결 방법
- 아키텍처 변경 이력

## vs Archive

`archive/` 폴더는 임시 보관 후 삭제 가능한 문서용입니다.
`history/` 폴더는 영구 보관이 필요한 문서용입니다.
