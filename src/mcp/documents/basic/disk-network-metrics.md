## 기본 조치 방법

### 디스크 부족 해결
1. 불필요한 파일 삭제
2. 로그 파일 정리
3. 스토리지 확장 검토

### 네트워크 문제 해결
1. 네트워크 연결 상태 확인
2. 대역폭 사용률 점검
3. 라우팅 테이블 확인

### I/O 복합 문제 해결
1. 시스템 I/O 모니터링
2. 병목 지점 식별
3. 인프라 최적화 검토
