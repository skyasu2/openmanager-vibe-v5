# 서버 트러블슈팅 가이드

## 일반적인 서버 문제 해결

### CPU 사용률 높음
- 프로세스 확인: `top`, `htop` 명령어 사용
- 리소스 집약적 프로세스 식별
- 필요시 프로세스 재시작 또는 종료

### 메모리 부족
- 메모리 사용량 확인: `free -h`, `vmstat`
- 메모리 누수 프로세스 식별
- 스왑 사용량 모니터링

### 디스크 공간 부족
- 디스크 사용량 확인: `df -h`
- 큰 파일 찾기: `du -sh /*`
- 로그 파일 정리 및 압축

### 네트워크 연결 문제
- 네트워크 상태 확인: `netstat -tuln`
- 포트 연결 테스트: `telnet`, `nc`
- 방화벽 설정 확인

## 모니터링 도구
- Nagios, Zabbix, Prometheus
- 로그 분석: ELK Stack
- 성능 모니터링: Grafana
