# 고급 장애 사례 분석

## 복합적 장애 상황

### 메모리 누수 + CPU 과부하
**증상:**
- 메모리 사용률 지속적 증가
- CPU 사용률 90% 이상 유지
- 응답 시간 급격히 증가

**해결 방법:**
1. 메모리 덤프 분석
2. 프로파일링 도구 사용
3. 애플리케이션 재시작
4. 코드 레벨 최적화

### 네트워크 분할 (Network Partition)
**증상:**
- 일부 서버 간 통신 불가
- 클러스터 상태 불일치
- 데이터 동기화 실패

**해결 방법:**
1. 네트워크 토폴로지 확인
2. 스위치/라우터 상태 점검
3. Split-brain 방지 설정
4. 수동 복구 절차 실행

### 디스크 I/O 병목
**증상:**
- 높은 iowait 시간
- 디스크 큐 길이 증가
- 애플리케이션 응답 지연

**해결 방법:**
1. iostat으로 I/O 패턴 분석
2. 디스크 사용량 최적화
3. SSD 업그레이드 고려
4. 캐싱 전략 개선
