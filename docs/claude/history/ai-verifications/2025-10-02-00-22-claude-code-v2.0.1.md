- CPU: 백그라운드 모니터링 오버헤드

- 메모리: 히스토리 인덱싱 필요

- CPU: 복잡한 상태 관리로 증가

   - 자동 가비지 컬렉션: 오래된 체크포인트 자동 삭제 (설정 가능한 보존 정책)

   - Idle-time 처리: 비중요 작업(분석 수집)을 유휴 시간에 처리

3. **캐싱 전략 강화**
