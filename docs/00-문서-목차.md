## ⚡ **빠른 참조 (Quick Reference)**

### 🚀 **5분 빠른 시작**
```bash
# 1. 프로젝트 설치
git clone <repository-url>
cd openmanager-vibe-v5

# 2. 의존성 설치
npm install

# 3. 개발 서버 시작
npm run dev

# 4. 브라우저에서 접속
http://localhost:3000
```

### 🔧 **자주 사용하는 명령어**
```bash
# 순차 서버 생성 리셋
curl -X POST localhost:3000/api/servers/next -d '{"reset":true}'

# AI 엔진 상태 확인
curl localhost:3000/api/ai-agent/optimized

# 빌드 및 배포
npm run build
vercel --prod

# 문제 해결
npm run clean && npm run build
```
