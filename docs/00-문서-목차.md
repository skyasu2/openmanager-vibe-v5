## 🚀 **빠른 참조**

### ⚡ **5분 빠른 시작**
```bash
git clone <repo>
cd openmanager-vibe-v5
npm install
npm run dev
# → http://localhost:3000
```

### 🔧 **문제 해결**
```bash
# 순차 생성 리셋
curl -X POST localhost:3000/api/servers/next -d '{"reset":true}'

# AI 엔진 상태 확인  
curl -X GET localhost:3000/api/ai-agent/optimized

# 빌드 문제 해결
npm run clean && npm run build
```

### 📊 **핵심 성과**
- **API 성능**: 3초 → 0.1초 (95% 개선)
- **메모리 효율**: 1.8GB → 300MB (84% 절약)
- **사용자 경험**: 순차 애니메이션으로 자연스러운 배포 시뮬레이션
- **Vercel 최적화**: 타임아웃 없는 안정적 실행
