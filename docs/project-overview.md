# 📋 OpenManager Vibe v5 - 프로젝트 개요

> **2모드 전용 통합 AI 엔진 시스템** - 2025년 7월 마무리 단계

## 🎯 **프로젝트 비전**

OpenManager Vibe v5는 **명확하고 단순한 2모드 AI 시스템**을 통해 서버 관리와 개발 작업을 혁신적으로 지원하는 통합 플랫폼입니다. 복잡한 AUTO 모드를 제거하고 LOCAL과 GOOGLE_ONLY 두 모드만으로 최적의 성능과 사용성을 제공합니다.

## 🚀 **핵심 특징**

### **🏠 LOCAL 모드 - 완전한 로컬 처리**

- **RAG + NLP + MCP + 하위 AI 엔진** 통합
- **완전한 프라이버시**: 모든 처리가 로컬에서 수행
- **네트워크 독립**: 인터넷 연결 없이도 동작
- **응답 시간**: 2-8초 (복잡도별 최적화)

### **🚀 GOOGLE_ONLY 모드 - Google AI + 선택적 로컬**

- **Google AI Studio** 메인 추론 엔진
- **선택적 로컬 처리**: 필요한 부분만 로컬 엔진 활용
- **빠른 응답**: 1-3초 클라우드 처리
- **고급 추론**: 복잡한 분석과 창의적 문제 해결

## 🏗️ **시스템 아키텍처**

### **UnifiedAIEngineRouter v3.3.0**

```
🎛️ 2모드 전용 통합 AI 엔진 라우터
├─ LOCAL 모드: RAG + NLP + MCP + 하위 AI 엔진
├─ GOOGLE_ONLY 모드: Google AI + 로컬모드(필요한 부분만)
├─ AUTO 모드: 완전 제거 (복잡성 제거)
└─ 명확한 처리 경로: 모드별 최적화된 파이프라인
```

### **주요 구성 요소**

#### **1. 통합 AI 엔진들**

- **Supabase RAG Engine**: 자연어 처리 및 벡터 검색
- **OptimizedKoreanNLPEngine**: 5단계 병렬 한국어 처리
- **Google AI Studio**: Gemini 기반 고급 추론
- **MCP Context Collector**: 실시간 시스템 정보 수집
- **Custom Engines**: 특화된 커스텀 처리

#### **2. 프론트엔드 시스템**

- **AISidebarV2**: 도메인 분리 아키텍처 적용 (1296줄)
- **AI 기능 패널**: 8개 AI 기능 아이콘 패널
- **실시간 차트**: 4개 게이지 + 성능 모니터링
- **모던 UI**: 애니메이션 효과 및 반응형 디자인

#### **3. 개발 도구**

- **TDD 시스템**: 테스트 주도 개발 환경
- **바이브 코딩**: Cursor AI + Claude Sonnet 3.7
- **성능 모니터**: 실시간 성능 모니터링 대시보드
- **문서 자동화**: AI 기반 문서 생성

## 🛠️ **기술 스택**

### **프론트엔드**

- **Next.js 14**: React 기반 풀스택 프레임워크
- **TypeScript**: 타입 안전성 확보 (0개 오류)
- **Tailwind CSS**: 유틸리티 기반 스타일링
- **Framer Motion**: 애니메이션 및 인터랙션

### **백엔드**

- **Node.js**: 서버 사이드 런타임
- **Supabase**: 데이터베이스 및 RAG 엔진
- **Google AI Studio**: Gemini API 통합
- **MCP Protocol**: Model Context Protocol

### **배포 환경**

- **Vercel**: 메인 웹앱 호스팅 (무료 티어 최적화)
- **GCP e2-micro VM**: MCP 서버 (<http://104.154.205.25:10000/>)
- **무료 티어**: 완전한 무료 운영 환경 구축
- **암호화**: 환경변수 보안 관리

### **개발 방법론**

- **바이브 코딩**: AI 협업 개발 (30분 개발 + 5분 AI 검토)
- **TDD**: 테스트 주도 개발 (482개 테스트 100% 통과)
- **코드 품질**: ESLint/Prettier로 A등급 달성
- **Vercel 배포**: 자동화된 CI/CD
