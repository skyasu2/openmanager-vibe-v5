# Temp 디렉터리

**목적**: 임시 파일, 작업 중인 문서

**현재 상태**: 비어있음 (정리 완료)

---

## 📝 사용 규칙

### 허용 파일

1. **작업 중인 초안** (24시간 이내)
2. **테스트 결과물** (일시적)
3. **임시 분석 보고서** (검증 전)

### 정리 규칙

- **24시간 초과**: 적절한 디렉터리로 이동
- **완료된 문서**: `../analysis/` 또는 `../archive/`로 이동
- **불필요한 파일**: 즉시 삭제

---

## 🗂️ 이동 대상 디렉터리

| 문서 유형   | 이동 위치      |
| ----------- | -------------- |
| 분석 보고서 | `../analysis/` |
| 아카이브    | `../archive/`  |
| 계획 문서   | `../planning/` |
| 스펙 문서   | `../specs/`    |
| 테스트 결과 | `../testing/`  |

---

**⚠️ 주의**: 이 디렉터리는 항상 비어있거나 최소한의 파일만 유지해야 합니다.
