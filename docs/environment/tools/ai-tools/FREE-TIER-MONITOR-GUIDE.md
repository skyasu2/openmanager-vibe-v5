# 무료 티어 모니터 사용 가이드

**위치**: AI 사이드바 → AI 고급관리 아래 "무료 티어 모니터" 아이콘  
**목적**: Vercel, Supabase, Google AI 무료 티어 사용량 실시간 추적

---

## 🎯 접근 방법

### 1. AI 사이드바 열기

- 대시보드 우측 상단 "AI 어시스턴트" 버튼 클릭
- 또는 단축키 사용 (설정된 경우)

### 2. 무료 티어 모니터 선택

- AI 사이드바 우측의 아이콘 패널에서 **Activity 아이콘** (📊) 클릭
- 자동으로 "인사이트" 탭으로 전환되며 상단에 모니터 표시

---

## 📊 표시 정보

### Vercel

- **대역폭 사용량**: 현재 사용량 / 100GB
- **진행률**: 퍼센트 바 (녹색/노란색/빨간색)
- **상태**: 60% 이상 노란색, 80% 이상 빨간색

### Supabase

- **DB 용량**: 현재 사용량 / 500MB
- **진행률**: 퍼센트 바
- **상태**: 실시간 업데이트

### Google AI

- **API 호출**: 현재 사용량 / 1200 요청/일
- **진행률**: 퍼센트 바
- **경고**: 80% 도달 시 자동 알림

## ⚠️ 경고 시스템

### 노란색 경고 (60-80%)

- 사용량이 제한의 60%를 초과한 경우
- 모니터링 강화 권장

### 빨간색 경고 (80%+)

- 사용량이 제한의 80%를 초과한 경우
- 즉시 조치 필요
- 자동 캐싱 강화 활성화

### Google AI 특별 경고

```
⚠️ Google AI 사용량 주의
일일 제한의 80%에 도달했습니다.
캐싱이 자동으로 강화됩니다.
```

## 🔄 업데이트 주기

- **자동 갱신**: 60초마다
- **수동 갱신**: 페이지 새로고침 또는 아이콘 재클릭
- **캐시**: 1분 TTL

## 📞 문제 해결

### 데이터가 표시되지 않음

1. `/api/ai/cache-stats` 엔드포인트 확인
2. 브라우저 콘솔에서 에러 확인
3. 네트워크 탭에서 API 응답 확인

### 사용량이 0으로 표시됨

- Google AI API를 아직 사용하지 않은 경우 정상
- 첫 번째 AI 쿼리 후 업데이트됨

### 경고가 표시되지 않음

- 80% 미만인 경우 정상
- 임계값 도달 시 자동 표시
