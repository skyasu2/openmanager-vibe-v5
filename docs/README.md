## 🛠️ **개발 도구 빠른 참조**

### 🚀 **핵심 npm 스크립트**

#### **일일 사용 스크립트**

```bash
# 환경 설정
npm run mcp:perfect:setup        # MCP 자동 설정
npm run mcp:status               # MCP 상태 확인

# 개발 및 검증
npm run dev                      # 개발 서버 시작
npm run validate:quick           # 빠른 검증
npm run test:quality             # 코드 품질 검사

# 배포
npm run deploy:safe              # 안전한 배포
```

#### **문제 해결 스크립트**

```bash
# MCP 문제 해결
npm run mcp:perfect:setup        # MCP 재설정
# Cursor 재시작 필요

# 테스트 문제 해결
npm run test:unit                # 단위 테스트
npm run test:integration         # 통합 테스트

# 성능 문제 해결
npm run build:analyze            # 번들 분석
npm run perf:vitals              # 성능 측정
```
