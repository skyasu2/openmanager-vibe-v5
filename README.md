# 🚀 OpenManager Vibe v5

> **🏆 바이브 코딩 경연대회 준비 완료** - 차세대 AI 기반 서버 모니터링 플랫폼 ✨

## 📋 프로젝트 개요

OpenManager Vibe v5는 **오픈소스 AI 라이브러리**와 **혁신적인 통합 기술**을 활용하여 구축된 차세대 서버 모니터링 플랫폼입니다. 11개의 AI 엔진을 통합하여 지능형 서버 분석, 예측, 이상 탐지 기능을 제공합니다.

## ✨ 핵심 기능

### 🧠 **통합 AI 엔진 시스템 (v5.36.0)**

- **11개 AI 엔진 완전 통합**: 6개 오픈소스 + 5개 커스텀 엔진
- **마스터 AI 엔진**: 지능형 라우팅 및 자동 폴백 시스템
- **사고과정 로그 시스템**: 실시간 AI 추론 과정 가시화
- **성능 최적화**: 메모리 50% 절약, 응답시간 50% 향상

### 🔄 **AI 에이전트 마이그레이션 완료**

- **유닥 로그 → 사고과정 로그 변환**: 기존 사용자 행동 데이터 활용
- **패턴 제안 → Enhanced + Prediction 엔진 연동**: 지능형 패턴 학습
- **A/B 테스트 → Hybrid 엔진 실험 관리**: 다중 전략 테스트
- **성능 모니터링 → 11개 엔진 통합 대시보드**: 통합 성능 관리

## 🔧 설치 및 실행

```bash
# 저장소 클론
git clone <repository-url>
cd openmanager-vibe-v5

# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env.local
# .env.local 파일을 편집하여 필요한 환경 변수 설정

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
npm start
```
